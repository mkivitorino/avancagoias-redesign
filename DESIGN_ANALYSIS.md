# Análise de Design - Imagem de Referência

## Paleta de Cores
- **Azul Primário**: #003D82 (azul escuro/marinho)
- **Amarelo Primário**: #FFD700 (amarelo ouro/vibrante)
- **Laranja Secundário**: #FF9900 (laranja vibrante)
- **Branco**: #FFFFFF (fundo)
- **Cinza Claro**: #F5F5F5 (fundo secundário)
- **Azul Claro**: #0066CC (azul mais claro)

## Tipografia
- **Fontes**: Aparenta usar fontes sans-serif modernas (possível: Arial, Helvetica, ou similar)
- **Estilos**: Negrito para títulos e destaques, regular para corpo
- **Tamanhos**: Títulos grandes (36-48px), subtítulos (24-28px), corpo (14-16px)

## Elementos de Design
1. **Fitas/Banners**: Elementos em forma de fita com cores vibrantes (azul, amarelo, laranja)
2. **Cards/Proposições**: Dispostos em grid com sombra suave
3. **Botões**: Retangulares, com cores vibrantes (amarelo "SOU A FAVOR", azul "SOU CONTRA")
4. **Ícones**: Estilo simples, cores sólidas
5. **Espaçamentos**: Generosos, com muita respiração visual
6. **Elementos Geométricos**: Formas angulares, diagonais, criando movimento

## Layout
- **Seções**: Bem definidas com espaçamento vertical generoso
- **Alinhamento**: Centralizado com elementos fluindo naturalmente
- **Hierarquia**: Títulos grandes, subtítulos médios, corpo pequeno
- **Imagens**: Integradas com conteúdo, não apenas background

## Componentes Principais
1. **Header**: Logo + navegação simples
2. **Hero Section**: Imagem grande com texto sobreposto
3. **Proposições**: Cards em grid com votação
4. **Call-to-Action**: Botões destacados em cores vibrantes
5. **Newsletter**: Seção de inscrição
6. **Footer**: Informações e links

## Funcionalidades a Manter
- Sistema de votação (SOU A FAVOR / SOU CONTRA)
- Filtros por eixo temático
- Paginação de ideias
- Autenticação OAuth
- Envio de ideias
- Página administrativa
- Formulário Typeform em /escuta
- Sistema de notificações por email
- Facebook Pixel
